Ethereum Derivative Builder

            

Enter The Terms of a Derivatives-Style Contract

Our templates allow you and a counterparty to create a smart contract that behaves like a derivative.
First fill out the details of your contract and compile your Solidity code.

Cut and Paste Your Solidity Code Into the Online Solidity Compiler:

Copy and paste the code into this online Solidity Complier. Hit enter. It will convert your Solidity code into, among other things, Web3 Code. Copy that code.

Copy and Paste the Web3 Code into a .js File

Take the resulting code and save it as a .js file in the same directory that you run Geth from.  

Open the Javascript Console in Geth

Log into Geth's Javascript Console using: 
  console
Now enter this command, inserting your file name: 
  loadScript('NameOfFile.js');
As soon as someone mines a block (it may take a few seconds), your contract will exist on the blockchain!
